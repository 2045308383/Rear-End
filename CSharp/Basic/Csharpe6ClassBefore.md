#### C# 变量
> C#是一门纯面向对象的,本质上任何一种数据类型都是一个对象不想C++,Java 有值类型,引用类型 
* 变量的初始化问题
  * 避免空指针异常 例如：
  ```C#
   Person li=new;
   
   String name=li.UserName; //这样的错误经常犯,有时候代码相隔太远,或者是忘了判断 就会导致空指针异常
  ```
###### 建议:多使用空值传播运算符使得代码更加简单: name=li?.UserName; 对于int数值类型 也可以使用可null类型 int age?=li>.Age;
