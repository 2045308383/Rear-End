NET Core 命令行接口 (CLI) 工具
----
### `Dotnet Build` 
> 生成项目及其所有依赖项。
* dotnet build 命令将项目及其依赖项生成为一组二进制文件。 二进制文件包括中间语言 (IL) 文件（带 `.dll 扩展名`）和用于调试的符号文件（带 `.pdb 扩展名`）中的项目的代码。 生成`依赖项 JSON 文件 (\*.deps.json)`，该文件列出了`应用程序的依赖项`。 生成 `\*.runtimeconfig.json 文件`，该文件指定应用程序的共享运行时及其版本。
